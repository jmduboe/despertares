# Environment Variables for Despertares Monorepo

## Backend (.env)
# DATABASE_URL="file:./dev.db" (development) | "postgresql://..." (production)
# JWT_SECRET="your-secret-key" (min 32 chars in production)
# JWT_EXPIRES_IN="7d"
# FRONTEND_URL="http://localhost:3000" (development) | "https://yourdomain.com" (production)
# PORT=3001
# NODE_ENV="development" | "staging" | "production"

## Frontend (.env)
# NEXT_PUBLIC_API_URL="http://localhost:3001" (development) | "https://your-api-domain.com" (production)
# NODE_ENV="development" | "staging" | "production"

## Security Notes:
# - Never commit .env files to version control
# - Use strong JWT secrets in production (min 32 characters)
# - Use PostgreSQL in production (not SQLite)
# - Set CORS properly in production
# - Use HTTPS URLs in production

## Database Setup:
# Development: SQLite (file:./dev.db)
# Staging: PostgreSQL
# Production: PostgreSQL with proper connection string

## Deploy URLs Examples:
# Frontend: https://despertares.vercel.app
# Backend: https://despertares-api.railway.app
# Database: Railway/Supabase PostgreSQL
